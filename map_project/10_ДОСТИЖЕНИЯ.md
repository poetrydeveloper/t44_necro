# Система достижений

## Типы достижений
| Тип | Условие | Награда |
|-----|---------|---------|
| Убийства | 100/500/1000 врагов | Золото + Титул |
| Коллекция | 50/200/500 карточек | Эссенция |
| Мерж | 10/50/100 слияний | Редкая карточка |
| Уровень | 10/25/50/100 уровень | Усиление |
| Дуэли | 5/25/100 побед | Скин/Пет |
| Забеги | 1/5/10/50 Run-ов | Эксклюзивный предмет |

## Титулы (награда за ачивки)
- "Могильщик" (100 убийств)
- "Повелитель мертвых" (1000 убийств)
- "Коллекционер" (500 карточек)
- "Чемпион арены" (100 побед в дуэлях)

## Отслеживание
- Автоматическое через сервер
- При получении - уведомление на UI
- Прогресс виден в меню достижений
