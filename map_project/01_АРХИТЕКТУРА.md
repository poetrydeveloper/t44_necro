# Архитектура проекта Necromancer Roguelite

## Технологический стек
- **Движок:** Roblox Studio
- **Синхронизация:** Rojo + Git
- **Язык:** Luau (с типизацией)
- **Сохранение:** DataStoreService

## Структура папок (src/)
src/
├── client/ # Клиентская часть (игроки)
│ ├── controllers/ # Контроллеры (движение, бой, ритуал)
│ ├── managers/ # Менеджеры (UI, камера, ввод)
│ ├── combat/ # Боевые системы (авто-тайминг, снаряды)
│ └── ui/ # Интерфейс (HUD, панели, экраны)
│
├── server/ # Серверная часть
│ ├── services/ # Главные сервисы (бой, коллекция, прогресс)
│ ├── managers/ # Менеджеры (игроки, враги, трупы)
│ ├── combat/ # Расчет урона, мана-система
│ ├── world/ # Генерация мира и дорог
│ └── progression/ # Система уровней и опыта
│
├── server-modules/ # Модули для сервера
│ ├── collection/ # Карточки, вес душ
│ ├── ritual/ # Ритуал (мерж, мутации)
│ ├── duel/ # PvP дуэли
│ └── achievement/ # Достижения и титулы
│
├── shared/ # Общее для клиента и сервера
│ ├── types/ # Типы данных
│ ├── enums/ # Перечисления
│ ├── utils/ # Вспомогательные функции
│ └── helpers/ # Хелперы (события, валидация)
│
├── constants/ # Конфиги и баланс
│ ├── balance/ # Игровой баланс
│ ├── units/ # Данные юнитов
│ ├── items/ # Данные предметов
│ ├── world/ # Данные локаций
│ ├── recipes/ # Рецепты мержа и мутаций
│ └── achievements/ # Список достижений
│
└── data/ # Импорт данных
├── importers/ # Импорт из Excel
└── tables/ # CSV/JSON таблицы

## Ключевые системы
1. **Auto-комбат** - автоматические заклинания и юниты
2. **Коллекция карточек** - вес душ, лимиты, мерж
3. **PvP дуэли** - вызов на бой, награды
4. **Прогрессия** - уровни, опыт, скиллы
5. **Сохранение** - DataStore с авто-сейвом
